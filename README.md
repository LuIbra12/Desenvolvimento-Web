# Desenvolvimento-Web
Na atividade do lab16 o professor requisitou que eu fizesse uma calculadora unindo as 3 linguagens de programação que estou estudando em sua disciplina, html, css e java script. Para isso fiz o uso do html para conter a estrutura básica da calculadora, o css para estilizar a calculadora e o java script para capturar as entradas e operações aritméticas da calculadora.
